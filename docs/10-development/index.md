# Development

The Development category covers the core disciplines of building software — from mobile apps to web applications, from frontend interfaces to backend systems, and the full-stack integration that ties it all together.

## Topics

| Topic | Description |
|-------|-------------|
| [Mobile Development](mobile-development) | Android, iOS, and cross-platform frameworks (Flutter, React Native) |
| [Web Development](web-development) | HTML5, CSS3, JavaScript, responsive design, PWAs, and web security |
| [Backend Development](backend) | REST APIs, GraphQL, databases, authentication, caching, and scalability |
| [Frontend Development](frontend) | React, Vue, Angular, state management, component patterns, and performance |
| [Full Stack Development](full-stack) | End-to-end architecture, deployment, DevOps, and system integration |

## Overview

- **Mobile Development** — Build native and cross-platform apps for billions of users across Android and iOS
- **Web Development** — Master the three pillars: HTML, CSS, and JavaScript, plus modern APIs and performance
- **Backend Development** — Design scalable APIs, manage data, handle authentication, and build resilient systems
- **Frontend Development** — Create interactive UIs with modern frameworks, optimize performance, and ensure accessibility
- **Full Stack Development** — Bridge frontend and backend to deliver complete, deployable applications
