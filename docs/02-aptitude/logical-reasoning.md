# Logical Reasoning Patterns

{: .note }
Logical reasoning tests your ability to think systematically, identify patterns, and draw valid conclusions from given statements.

## Introduction

Logical reasoning refers to the ability to think systematically, analyze information, identify patterns, and draw valid conclusions. It involves deductive and inductive reasoning, pattern recognition, and critical thinking skills.

**Why It Matters:**
- Core component of most aptitude tests
- Measures analytical and critical thinking
- Essential for roles requiring problem-solving
- Used in consulting and technology interviews

## Theory Notes

### Series Completion

**Number Series Patterns:**
1. **Arithmetic Progression**: Constant difference (2, 5, 8, 11, 14, ?)
2. **Geometric Progression**: Constant ratio (2, 6, 18, 54, ?)
3. **Square/Cube Series**: Terms are squares or cubes
4. **Fibonacci-like**: Each term is sum of previous terms
5. **Alternating Series**: Two different patterns interleave

**Letter Series Patterns:**
1. **Alphabetical order**: Consecutive letters (A, C, E, G, ?)
2. **Reverse order**: Backward alphabetical (Z, X, V, T, ?)
3. **Alternating patterns**: Two interleaved sequences

### Coding-Decoding

**Letter Coding:**
- Direct coding: Each letter replaced by another (A→D, B→E, C→F)
- Reverse coding: Alphabet reversed (A→Z, B→Y, C→X)
- Position-based coding: Based on alphabetical position

**Decoding Techniques:**
1. Compare coded and actual words
2. Identify pattern of change
3. Apply pattern to decode
4. Verify with multiple examples

### Blood Relations

**Key Relationships:**
- Father, Mother, Son, Daughter
- Brother, Sister
- Uncle, Aunt (Paternal/Maternal)
- Nephew, Niece
- Grandfather, Grandmother

**Problem-Solving Approach:**
1. Draw family tree
2. Mark known relationships
3. Use given clues
4. Determine unknown relationships
5. Verify the complete tree

### Direction Sense

**Cardinal Directions:**
- North (N), South (S), East (E), West (W)
- North-East (NE), North-West (NW)
- South-East (SE), South-West (SW)

**Shadow Problems:**
- Morning (sun in East): Shadow points West
- Evening (sun in West): Shadow points East
- Noon: Shadow points North (Northern hemisphere)

### Syllogisms

**Basic Structure:**
- Major Premise: General statement
- Minor Premise: Specific statement
- Conclusion: Logical deduction

**Venn Diagram Approach:**
1. Draw circles for each category
2. Mark regions based on statements
3. Check if conclusion follows
4. Verify with all possible diagrams

### Seating Arrangements

**Types:**
- **Linear**: People seated in a row
- **Circular**: People seated around a table
- **Double Row**: Two rows facing each other

**Problem-Solving Approach:**
1. Fix one person's position
2. Place others relative to fixed person
3. Use given clues
4. Handle multiple possibilities
5. Verify final arrangement

## Best Practices

1. **Identify patterns immediately**: Look for arithmetic, geometric, or other progressions
2. **Use process of elimination**: Rule out wrong answers systematically
3. **Draw diagrams for spatial problems**: Visualize directions and seating
4. **Test assumptions**: Verify your reasoning with examples
5. **Practice systematically**: Work through each topic type
6. **Build speed gradually**: Accuracy first, then speed
7. **Review mistakes**: Understand why you got something wrong
8. **Learn from solutions**: Study correct approaches

## Common Mistakes to Avoid

| Mistake | Impact | Solution |
|---------|--------|----------|
| Not identifying patterns | Can't solve series | Calculate differences between terms |
| Confusing directions | Wrong final position | Draw directional diagrams |
| Not drawing family trees | Confusion in blood relations | Always draw the tree |
| Rushing through puzzles | Missing key information | Read all conditions carefully |
| Not verifying answers | Incorrect solutions | Check against all conditions |

## Quick Reference

### Series Patterns
- Arithmetic: Constant difference
- Geometric: Constant ratio
- Square/Cube: Perfect squares/cubes
- Fibonacci: Sum of previous terms
- Alternating: Two interleaved sequences

### Direction Rules
- Clockwise: N → E → S → W → N
- Anticlockwise: N → W → S → E → N
- Shadow morning: Points West
- Shadow evening: Points East

### Syllogism Patterns
- All A are B, All B are C → All A are C
- Some A are B, All B are C → Some A are C
- No A are B, All B are C → No A are C
