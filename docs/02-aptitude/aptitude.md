# General Aptitude Preparation

{: .note }
Aptitude tests are used by 70%+ of companies as initial screening. Master numerical reasoning, verbal ability, and logical thinking.

## Introduction

General aptitude refers to the broad set of cognitive abilities that indicate a person's potential to learn, adapt, and perform in various situations. Aptitude tests measure numerical reasoning, verbal ability, logical thinking, and problem-solving skills.

**Why Aptitude Tests Matter:**
- Used by 70%+ of companies as initial screening
- Measures raw cognitive ability, not just learned skills
- Predicts job performance across various roles
- Standardizes evaluation across candidates

## Aptitude Test Categories

| Category | Focus | Key Skills |
|----------|-------|------------|
| Numerical Reasoning | Math, data interpretation | Calculations, percentages, ratios |
| Verbal Reasoning | Reading comprehension, grammar | Vocabulary, sentence structure |
| Logical Reasoning | Pattern recognition, critical thinking | Series, coding-decoding, puzzles |
| Abstract Reasoning | Visual patterns, spatial reasoning | Pattern recognition, mental rotation |
| Situational Judgment | Work-related scenarios | Decision-making, problem-solving |

## Core Topics

### Numerical Reasoning
- **Percentages**: Calculating percentages, percentage change
- **Profit & Loss**: Cost price, selling price, profit/loss percentage
- **Ratio & Proportion**: Simple ratio, compound ratio
- **Time & Work**: Individual work rates, combined work
- **Time & Distance**: Speed, time, distance, relative speed
- **Simple & Compound Interest**: Principal, rate, time

### Verbal Reasoning
- **Reading Comprehension**: Understanding passages, inferring meaning
- **Grammar**: Tenses, subject-verb agreement, articles
- **Vocabulary**: Synonyms, antonyms, word meanings
- **Sentence Correction**: Identifying and fixing errors

### Logical Reasoning
- **Series Completion**: Number series, letter series
- **Coding-Decoding**: Letter coding, number coding
- **Blood Relations**: Family tree problems
- **Direction Sense**: Navigation and direction problems

## Key Formulas

| Formula | Application |
|---------|-------------|
| Percentage Change = (New-Old)/Old × 100 | Growth rates, comparisons |
| Profit % = (Profit/CP) × 100 | Business profitability |
| Speed = Distance/Time | Travel problems |
| Work = Rate × Time | Project completion |
| SI = P × R × T / 100 | Basic interest calculation |
| CI = P(1 + R/100)^T - P | Compound growth |

## Time Management Strategies

### Time Allocation
- **30 questions/30 min**: 1 minute per question
- **40 questions/45 min**: ~1.1 minutes per question
- **50 questions/60 min**: 1.2 minutes per question

### Question Selection Strategy
1. Scan all questions first (2 minutes)
2. Identify easy/medium/hard problems
3. Start with easy questions (build confidence)
4. Tackle medium questions next
5. Attempt hard questions last
6. Review skipped questions

## Best Practices

1. **Start Early**: Begin preparation 6-8 weeks before the test
2. **Practice Daily**: Consistent practice beats cramming
3. **Time Yourself**: Always practice under timed conditions
4. **Learn Shortcuts**: Master shortcut formulas for speed
5. **Build Mental Math**: Practice calculations without calculator
6. **Focus on Weaknesses**: Spend extra time on weak areas
7. **Take Mock Tests**: Simulate real test conditions regularly
8. **Review Mistakes**: Analyze errors to avoid repetition
9. **Read Regularly**: Improve verbal skills through reading
10. **Stay Positive**: Confidence improves performance

## Common Mistakes to Avoid

| Mistake | Impact | Solution |
|---------|--------|----------|
| Spending too much time on one question | Miss easier questions | Set time limits, move on if stuck |
| Not practicing under timed conditions | Unprepared for real test | Always practice with time limits |
| Ignoring weak areas | Consistent poor performance | Identify and focus on weaknesses |
| Using calculator during practice | Doesn't build mental math | Practice mental calculations |
| Not reviewing mistakes | Don't learn from errors | Analyze every mistake |

## Quick Reference

### Key Formulas
- Percentage Change: (New-Old)/Old × 100
- Profit %: Profit/CP × 100
- Average Speed: 2xy/(x+y) for equal distances
- Time & Work (together): xy/(x+y) days
- Simple Interest: PRT/100
- Compound Interest: P(1+R/100)^T - P

### Time Management Rules
- Skip if stuck > 1.5 minutes
- Return to skipped questions at end
- Read questions before options
- Eliminate obviously wrong options
