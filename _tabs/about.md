# About This Guide

A comprehensive interview preparation resource covering **120+ topics** across every aspect of technical interviews at FAANG, top startups, and MNCs.

---

## What Makes This Guide Different

| Feature | Description |
|---------|-------------|
| **Structured Learning Paths** | Follow a 12-week roadmap from beginner to interview-ready |
| **Progress Tracking** | Track which topics you've completed, saved locally in your browser |
| **120+ Deep-Dive Topics** | Each topic has theory, practice questions, cheat sheets, and flash cards |
| **Real FAANG Problems** | Actual interview questions from Google, Amazon, Meta, Apple, Microsoft |
| **Multiple Formats** | Tables, diagrams, code examples, mind maps, and Mermaid flowcharts |

## Role-Based Learning Paths

### Backend Engineer
Resume, DSA, OS, DBMS, Computer Networks, System Design, SQL, Java/Python, Docker, Kubernetes, API Design

### Frontend Engineer
Resume, DSA, JavaScript, React, CSS/HTML, Web Development, System Design, Testing, Git, Performance Optimization

### Data Scientist / ML Engineer
Resume, Python, SQL, Statistics, Machine Learning, Deep Learning, NLP, LLMs, RAG, MLOps, Data Analytics

### DevOps / SRE Engineer
Resume, Linux, Networking, Docker, Kubernetes, Terraform, CI/CD, AWS/Azure, Monitoring, Security, SRE

### Full Stack Developer
Resume, DSA, JavaScript, React, Node.js, Python, SQL, System Design, Docker, CI/CD, Testing
