## Overview

Flash cards are an effective learning tool using active recall and spaced repetition to improve long-term retention of information.

## Key Areas

### 1. Card Types
- Question and answer
- Definition cards
- Code snippets
- Concept explanations

### 2. Creation Guidelines
- One concept per card
- Clear and concise
- Include examples
- Add visual cues

### 3. Review System
- Spaced repetition schedule
- Performance tracking
- Difficulty adjustment
- Regular updates

### 4. Digital Tools
- Anki
- Quizlet
- Physical cards
- Mobile apps

## Tips
- Create cards as you learn
- Review consistently
- Test both directions
- Update cards based on performance
